# To-Do List with Progress Tracker

## Project Overview
A simple and elegant to-do list application built with vanilla HTML, CSS, and JavaScript. Users can add, edit, delete, and mark tasks as complete with automatic progress tracking.

## Features
- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as complete
- 📊 Real-time progress bar showing completion percentage
- 💾 Persistent storage using localStorage (tasks saved locally)
- 🎨 Responsive and user-friendly UI

## Technologies Used
- **HTML** - Markup and structure
- **CSS** - Styling and layout
- **JavaScript** - Functionality and interactivity
- **localStorage** - Client-side data persistence

## How to Use
1. Open the application in your browser
2. Type a task in the input field
3. Click the "Add" button to add the task to the list
4. Click on a task to mark it as complete (strikethrough)
5. Click "Edit" to modify a task
6. Click "Delete" to remove a task
7. Watch the progress bar update as you complete tasks!

## Features Details
- **Add Tasks**: Enter text and click Add button
- **Edit Tasks**: Click the Edit button next to a task
- **Delete Tasks**: Click the Delete button to remove
- **Mark Complete**: Click the task text to toggle completion status
- **Progress Tracking**: Automatic percentage calculation and visual progress bar
- **Data Persistence**: All tasks are saved to browser's localStorage
